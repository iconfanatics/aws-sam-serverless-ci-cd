# AWS SAM Serverless CI/CD Project

This project demonstrates a production-ready serverless application
built with AWS SAM and deployed using GitHub Actions CI/CD.

## Tech Stack
- AWS Lambda (Python)
- API Gateway
- AWS SAM
- CloudFormation
- GitHub Actions

## CI/CD Flow
1. Push to `main` branch
2. GitHub Actions triggers
3. SAM build & deploy
4. Automatic deployment to AWS
